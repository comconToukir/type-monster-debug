What are the differences between localStorage and sessionStorage?

localStorage is persistent storage. Meaning, it will stay if the page reloads. On the other side sessionStorage is not persistent and will not survive page reloads.

What are the differences between global scope and local scope?

Global scope contains all of the things defined outside of all code blocks. A global variable has global scope. A global variable is accessible from anywhere in the code. Local scope contains things defined inside code blocks. A local variable has local scope. A local variable is only accessible where it's declared.

What is javaScript event loop?

In javascript to make asynchronous programming possible event loop is used. It is a cyclic system where call stack is used to keep track of all operations in line to be executed and event queue is used to send new functions to the stack for processing. Event loop constantly checks if the call stack is empty. If it is empty, new functions are added from the event queue. If it is not, then current function call is processed.

How many ways can we get undefined?

  • A variable with unassigned value.
  • Function with no return.
  • Parameter that is not passed will be undefined
  • If return has nothing on the right side.
  • Property that doesn't exist on an object will give undefined.
  • Accessing array elements outside of the index range.
  • Deleting an element inside an array.
  • Setting a value directly to undefined.
